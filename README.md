# Landing page: Fintech Puppy Park powered by BNY

A single-page, static site prepared for BNY's team at Booth 1039 at eMerge Americas 2026. The activation on the floor is the Fintech Puppy Park, a warm icebreaker; the page speaks to the real audience, which is BNY's Eliza platform, AI enablement, and SPIN / Ascent partnership leads.

Built on BNY's own visual system: deep navy (#04243C) with a teal (#2B9CAE) accent, Söhne / Inter neutral grotesk, generous negative space, a subtle teal wave behind the hero, numbers-lead stat block in the BNY "20,000 Empowered Builders" cadence. No Eduba chrome anywhere above the footer. Bank-grade restraint: no gradients, no neon, no paw-print bullets, no consumer-fintech tropes.

The page carries the ICM methodology frame, the Correlation One case study (Pacific Life and Colgate-Palmolive), real anchors to the ICM repo and the Ethics Engine paper, and a CTA to a 30-minute call with Matt Creamer.
